Introduction

Protocol Overview

A Decentralized Exchange Focused on creating long term incentives through the use of a time staking mechanism that penalizes users for premature withdrawls and rewarding users with Platform Rewards in the form of NFTs.

Systems Overview

Sigmadex utilizes the EIP-2535 Diamond architecture to bring the native token SDEX, A multi-token farm, an autocompounding native token pool, and a Governance structure. Swap/Pool systems are coming soon.

API Spec

A description and overview of the various Sigmadex Smart Contract functions, enabling on-chain interaction by third party developers and other decentralized applications.
