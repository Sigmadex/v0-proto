Reduced Penalty

The reduced Reward penalty is onchain as the ReducedPenaltyReward, has internal state denoted bye the rP prefix and has its Reward interface defined in ReducedPenaltyFacet. When applied to a {UserPosition} or {VaultUserPosition} it protects the user in the event they feel emplored to withdraw before their timeEnd. In this event, a user will find their refund greater up to the reductionAmount provided in the NFT. This reductionAmount is derived from the value of the Reward when it was minted. For example, if the reward Amount was 10 USDT, it will offer up to a 10 USDT penalty reduction.
